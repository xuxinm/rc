Common configuration files
--------------------------

RC files
--------

* bashrc --> $HOME/.bashrc
* profile --> $HOME/.profile
* emacs --> $HOME/.emacs
* vimrc --> $HOME/.vimrc
* gitconfig --> $HOME/.gitconfig
* matplotlibrc --> $HOME/.matplotlibrc

Other files
-----------

safari.css : Enable fontsize tuning for Safari browser
pdfmerge.sh : Combines multiple PDF files
